# 🎯 Simple Quiz App

A responsive and interactive quiz application built with **HTML**, **CSS**, and **Vanilla JavaScript**. The app displays multiple-choice questions, instantly checks answers, keeps track of the user's score, and allows the quiz to be restarted after completion.

---

## 📖 About the Project

This project was created to practice **JavaScript DOM Manipulation**, **Events**, **Arrays & Objects**, **Functions**, and **Dynamic UI Rendering**.

The quiz presents one question at a time. When the user selects an answer:

- The selected answer is highlighted.
- The correct answer is revealed.
- All answer buttons become disabled.
- The **Next** button appears.
- After the last question, the final score is displayed with an option to play again.

- 📋 Multiple-choice quiz

## ⚙️ How It Works

1. The quiz starts automatically.
2. The first question is displayed.
3. The user selects an answer.
4. The selected answer is checked.
5. The correct answer is highlighted.
6. The Next button appears.
7. The process repeats until all questions are answered.
8. The final score is displayed.
9. The user can restart the quiz.
